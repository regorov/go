Package: github.com/kierdavis/go/mm3dmodel
==========================================

[doc](http://gopkgdoc.appspot.com/pkg/github.com/kierdavis/go/mm3dmodel)

Install
-------

    $ go get github.com/kierdavis/mm3dmodel
